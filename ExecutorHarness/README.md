# Executor Harness

This folder contains example scripts for testing with the StarLib Editor emulator.

## Usage

1. Open StarLib Editor
2. Click **Run > Open Emulator**
3. The emulator loads `StarLib.lua` and runs your generated script
4. View the rendered UI in the emulator canvas
5. Interact with buttons, toggles, sliders, etc.
6. Check the console panel for print/warn/error output

See `SampleScript.lua` for an example of a generated StarLib UI.
